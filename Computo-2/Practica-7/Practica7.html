Position: Para poder tener estructuras mas complejas, tenemos que discutir la propiedad position (posición). Esta propiedad tiene un montón de posibles valores, sus nombres no tienen sentido y son casi imposibles de memorizar. width: auto; La propiedad CSS width especifica la anchura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento. height: 100% Ocupa el Alto de capa que sea el de la ventana del navegador. Margin:; Las margin propiedades CSS se usan para crear espacio alrededor de elementos, fuera de cualquier borde definido. Padding Las paddingpropiedades de CSS se usan para generar espacio alrededor del contenido de un elemento, dentro de cualquier borde definido.
